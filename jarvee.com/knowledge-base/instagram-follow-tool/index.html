The Instagram Follow tool lets you automatically follow Instagram users based on various sources. This tool is useful when you want to reach new people in your niche, interact with them and also find business opportunities. Following other Instagram users that share your interest will also help you grow your account because if you have high-quality content you will not stay unnoticed.

To use the Instagram Follow tool, go to Tools tab and click the Follow action of  the account where you want to run the tool or click MORE TOOLS

You will then notice 4 tabs:

  1. Follow Settings
  2. Follow Sources
  3. Follow Results
  4. Extracted Users

1. Follow Settings

This is the main muscle of the tool. It has 4 sections: Run Timers, Follow Limits, User Filters and After Follow Actions.

Run Timers

On this section, you will need to set-up how and when the tool is going to execute the “follow” actions.

  • Start with the “wait between time” before each new follow operation
  • Select the number of users that you want to follow per operation and the delay or pause time between each follow
  • Set the hours when you want the tool to execute
  • If you want to temporarily stop the tool after executing a certain number of operations, you can opt to enable a “random sleep time”
  • You may also select to stop the tool after it has followed X users
  • You can choose the days of the week when the tool will execute actions

There is an option to rotate the days when the tool is going to operate.  If you enable this option, Jarvee would use the same number of days you have initially selected and rotate it.  Let’s say you checked Monday, Wednesday, and Friday, on the succeeding week, Jarvee will select random 3 days to execute the action.  You may also exclude weekends in the variation should you prefer not to follow on said days.

Increase actions per operation daily by X and Y until it reaches Z

  • This option will increase the number of follow actions between X  and Y amount of people per operation. The first value will increase the left minimum side of the above specified range the second value will increase the right maximum side of the above specified range until it reaches the Z amount you specify on the third field.

Follow Limits

1. This is the section where you set the daily limit with regard to the number of people you want to follow.  For instance, you want to follow between 200 to 300 people per day, the Follow tool will only do a maximum of X follows ranging from 200 to 300 on a daily basis.

  • You may choose to gradually increase the follow limit each day until it hits your preferred maximum count.

2. Enable contextual actions will make the follow action look more natural. This option works both the Embedded Browser and the API. For the API, the option to ‘Enable API Full Emulation’ must be checked as well under Advanced Profile Settings.

📌 Click here to know more about how contextual actions work for the Follow tool.

3. There is an option to enable automatic follow/unfollow, this is where you can schedule the tool to stop following when it reaches total X account followings and automatically start the unfollow action.

4. If you want to use the account to only scrape users and send the extracted users to other accounts or to own account, you can enable the ‘Send to extracted users option.

5. You can set the tool to use sources that have high follow back ratio more than the others by checking the option ‘Favour sources with high follow back ratio’

Things to remember:

  • Please note that when the ‘Send extracted users’ option is enabled, the account will not execute a follow action, it will only scrape users based on the follow sources and filters you’ve configured in the tool.
  • The ‘Send extracted users’ option is ideally used for slave accounts where such accounts will only scrape users and send them to the main accounts that will use the extracted users as their source (specific users).
  •  If you want to set up the account such that it will scrape users during nightmode or when it’s outside execution hours then do the following actions during execution hours, you can enable the option ‘Only outside execution hours.
  • You have the option to send the extracted users to own account or other accounts and it will be sent to the Specific Users source of the selected tool.

User Filters

  • You can think of this as the “quality control” section as this is where all scraped users are screened before Jarvee decides whether to follow or skip the user based on the parameters you set.
  • As you can see from the image below, there are several filters you can apply to carefully target the users you want to follow.  However, keep in mind that if the user fails any of the filters you set, it will skip them, hence strict filters would have an impact on the number of people you can follow per operation.
  • To counter the effect of using strict filters, be sure to use as many sources as you possibly can, of course, it has to be a source that is related to your niche.

Extracted Users

We have moved the Auto Send Scraped users results to other accounts in Follow Settings tab

If you enabled the option ‘Send to extracted users’ in the Follow Settings tab > Follow Limits section, then the scraped users will be sent to the Extracted Users tab.

From here, you will see the record of all scraped users and you can choose to send the users to specific accounts or split with specific accounts.

To send scraped results to own account that did the scrapping as well as to other accounts (click the ADD ACCOUNTS button to select the account/s that will receive the users) in real time, make sure to check the option ‘Automatically send scraped results to other accounts‘ before starting the Follow tool.

You can send the scraped users to the Specific Users Source of the following tools:

  • Own account
  • Split scraped results with other accounts(use this if you want accounts to not have the same scraped users)
  • Follow
  • Like
  • Comment
  • Like Comments
  • Story Viewer

Additionally, you can also send the extracted users to:

  • Contact tool (Use Extracted Users tab)
  • Repost tool in Share with Specific Users instead of post on the wall
  • Comment tool dynamic mentions
  • Campaign Tag Users

After Follow Actions

  • Just like it says, after a user is followed, you can select to do additional actions such as like and/or comment on user’s latest post and send a direct message
  • Included in this section is the feature that lets you auto-accept pending follow request (if the account running the Follow tool is set to Private) as well as the ability to remove some sources that are not up to par with your desired “follow back” ratio.
  • You also have the option to mute the user after follow to keep your feed well kept.
  • To increase your chance of getting followed back, you can opt to view the story of the user you just followed.
  • And if you prefer to keep a constant number of followings, you can choose to unfollow oldest followed user after follow.

Use the Copy Settings option if you want to replicate the settings (and sources) to other Instagram accounts

2. Follow Sources

Basically, this is the tab that Jarvee will use to search for people to follow.  For each “follow” operation, Jarvee will select a random source from the list you provided.  You can choose to prioritize other sources by assigning a higher selection rank.

Follow users by hashtag search

Jarvee will search for users who posted with the hashtag word you have indicated.

  • You can choose to scrape only users that posted in the last X-Y minutes
  • There is an option to follow users that interacted with the hashtagged posts as well as followers of the users who posted with the hashtags.
  • If you want to scrape only users of the top-rated posts of the specified hashtags, you may do so.
  • For multiple keywords use ‘,’ as a separator; ie. cake, dessert, cookies, cupcake

Follow by username/name/bio

Use this source to follow users who had the keyword either in their username, name or bio (For multiple keywords use ‘,’ as a separator, for example, cake, dessert, cookies, cupcake)

Follow followers/followings of target account

Jarvee will follow the followers and/or followings of specific usernames you put in this source.

  • If you want to target the followers of  the target account, just type in the username and click “Add Account Followers” button and it will be added in the designated table.
  • Otherwise, click “Add Account Followings” if you wish to follow the followings of the target account.
  • You can opt to follow both followers and followings of the target accounts should you prefer it that way.

Follow followers of own followers

Use this source if you want to follow the followers of your followers

  • You can set Jarvee to follow the most recent X followers of your friends
  • Note that a random account from your followers will be selected when using this source

Follow followings of own followers

  • You can set Jarvee to follow followings of own followers (A random account from your followings will be chosen)

Follow Specific Users

Jarvee will follow the usernames or user IDs you added in this source.  Don’t forget to click “Add Usernames” button.

  • You can set the tool to pick users in order, else a random user will be selected from the list.
  • There is an option to use the user filters on this source, hence if the user fails any of the filters you’ve set, it will be removed from the list and will not be followed.  Otherwise, if the user passes all the user filters in place, then it will be followed.
  • By default, when a user from the list is followed, it will be removed.  However, if you plan to follow, unfollow then follow the same users over and over, then you can enable the option to not remove the user from the list.

Follow using Geo Location

This source will search for users that tagged the specified location in their post.

Just type your target location in the select locations field and you will see the suggested locations based on the word you provided so you can choose the exact one.  When you tick the corresponding checkbox, it will add that location to the table right below it.

If you have a list of location ids, then you can manually add it to this source as well.

  • You can set Jarvee to follow the followers of users that were scraped using this source.
  • There is also an option to follow users that interacted with posts posted in any of the specified locations.
  • You can choose to scrape only those that posted in the last X-Y minutes.

Follow users that interacted with posts on target accounts

Jarvee will search for likers or commenters or ‘likers and commenters’ on the target account’s posts.

  • You can set it to follow only those users who commented on the last X minutes or those that commented with a specific word as well as those that have a minimum character in their comment.
  • There is an option to scrape only posts that were published in the last X minutes
  • Be sure not to forget clicking the “Add” button.

Follow users that interacted with posts on target posts

Similar to following users that interacted with posts in target accounts, this source will enable you to follow users that liked or commented or liked and commented on a specific post URL.

  • You can set it to follow only those users who commented on the last X minutes or those that commented with a specific word as well as those that have a minimum character in their comment.
  • There is an option to scrape only posts that were published in the last X minutes
  • Be sure to click the “Add” button after you entered the post URL

Follow users that tagged Specific Users

This source will scrape and follow users that tagged specific users on their posts, provided they pass all the filters you’ve specified.

Follow suggested users

Select this source if you want to follow those users suggested by Instagram.

Follow own story viewers

Use this source to follow users who view your stories.

Follow from Explore Page

Please note that the Instagram Explore Page is unique for each account.  It consists of posts you’ve liked and posts liked by people whom you’ve liked (image and video).

Follow from Explore Shopping Page

Use this source to follow users who post products and show in Explore Shopping Page.

3. Follow Results

Here, you will see the users you’ve followed including the date and time they were followed.

You can also see additional information such as:

  • Keyword – the source used when searching for the followed user
  • Followed Back – you will see “Yes” if the user followed you back, otherwise, it will show No
  • Unfollowed – will display Yes or No if the user has been unfollowed once you start running the Unfollow tool on the account.

If the user was unfollowed outside Jarvee, this will also show Yes once you update the ‘was unfollowed’ flag via “MORE ACTIONS…”

  • IsPrivate – if the user that was followed is a private account, you will see Yes and No if it is not
  • Link – the URL or account link of the user that was followed

4.  Extracted Users

If you enabled the option ‘Send to extracted users’ in the Follow Settings tab > Follow Limits section, then the scraped users will be sent to the Extracted Users tab.

From here, you will see the record of all scraped users and you can choose to:

  • You also have the options “Extract additional user details”.  Use this if you want to extract additional details like bio, has profile picture, is business and other details.
  • “Also extract user engagement, average likes, average comments and most recent post date”.  Use this if you want to extract the respective information as mentioned.
  • “Also extract public email, phone and address”.  Use this if you want to extract email, phone and address from extracted users.
