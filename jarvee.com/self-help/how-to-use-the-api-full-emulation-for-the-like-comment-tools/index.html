How To Use The API Full Emulation For The Like/Comment Tools

We understand that most of our users out there are not that tech-savvy, and that’s why we made this step by step guide on how to use the API Full Emulation for the Like/Comment Tools.

Below you will find the step by step guide that you must follow to take advantage of this update.

The first thing to mention, this will work only for the Like and Comment tools and it will only work with a Scrapper Account, that will be used to extract users and send them to the Main account.

This means that even with this update, you will need to use a slave account to extract users and send them to the Like and Comment tool sources of your Main account. Below, you will find a step by step guide that you can follow to set that up.

When using this new API update, please don’t use any other tools on your account unless they are supported with API emulation. Here are the tools that are supported for now:

– Follow,
– Unfollow,
– Like,
– Comment,
– DM.

If you want to use a specific tool, please ask our support if it’s supported or not yet and they will get back to you on that.

If you use a tool that is not supported, you will probably get blocked when using the supported tools as well. so, please follow this advice.

Also, make sure that you are using a high-quality proxy, preferably 4G or mobile proxies. If you are using low-quality proxies, you will probably get blocked.

Let’s start with a definition of the scrapper/slave accounts:

What is a Scrapper account and what is the use of it?

A scrapper account is just a name that we gave to the account that will be used to extract and scrape users and send them to the Main account (that you wish to grow).

  • Why we need a Scrapper account?

We do suggest to use a scrapper account because doing a lot of API calls might get the Main account blocked after the last updates that Instagram released.

So it would be better to configure the Main account to do fewer API calls as much as possible. That’s why we suggest using a dummy account that will be doing those API calls for the Main account.

So all you need to do is create a couple of new accounts and add them to Jarvee. Then use them as Slave/Scrapper accounts.

  • How to use the Scrapper account?

The sources you want to target by your Main account, instead of adding them on the Source tab of the Main account as you used to do, you will now add them to the Source tab of the Scrapper account, that way, the scrapper account will extract the users from those sources and send them to your Main account.

This way, the API calls needed to find the suitable users from the sources you target, will be done by the Scrapper account, so the Main account will be doing just the actions you desire.

To enable this API update for your Main account, you will need to enable API full emulation option on the Main account profile Advanced Settings by checking the option below:

To do so, please go to the SOCIAL PROFILE tab, click on your Main account, then scroll down and check the option Show Advanced Profile Settings. Then make sure to check that option above.

Once you checked that option, please go to Social Profiles > select the account > click Actions On Selected Profiles and then click Re-Login Accounts as shown in the screenshot below (You need to do this even if your account have a valid status):

When it’s done with re-login (you’ll see the successful re-login on the Dashboard), please make sure to work only with the supported tools we mentioned at the beginning of this tutorial and make sure to have a Slave account that sends users to the Main account Like specific posts based on URL source.

To be able to scrape posts all you need to do is to use the Send to Extracted Posts option we added on the LIKE tool Settings tab of your Slave account.

Below, you will find a step by step guide showing you what to do to set your Scraper account to extract specific posts URLs for your Main account to Like or Comment on them.

1- Scrapper account configurations:

To be able to scrape posts all you need to do is to use the Send to Extracted Posts option we added on the LIKE tool Settings tab of your Scrapper account.

Please go to your Like tool Settings tab of your Scrapper/Slave account and check the option below:

By checking the option above, the Scrapper account won’t like the posts, from the sources you add on the Source tab, Instead, the account will extract the posts and add them to the Extracted Posts Tab.

Now, please go to the Like Sources tab of your Scrapper account and add all the sources you want to scrape users from (hashtags, Like Posts of Followers/following of other accounts…)

After that, go to the Extracted Posts tab of your Scrapper account, you can check the option to send the extracted posts automatically to your Main account.

To do that, go to the Like tool Extracted Posts tab and set it as indicated below:

Click on ADD ACCOUNT button, select the Main account where you want to send the extracted posts then click on OK

2- Main account configurations:

Once you do all that, you will need to go to your Main account Like tool Sources tab and Comment tool Sources tab and check the sources below where you will find the extracted posts. See screenshots below:

The main account Like tool –> Sources tab and check the following source:

The main account Comment tool –> Sources tab and check the following source:

Now To make sure your Main account is going to be using only the API, please do the following:

Please go to the SOCIAL PROFILE tab, click on your account, then scroll down and check the option Show Advanced Profile Settings.

–  Check the option highlighted in yellow.
–  Uncheck the options highlighted in blue.

Also, go to SETTINGS –> Social Platform –> Instagram Tab The make sure to uncheck the options below:

Once you do that, start the Like tool on the slave account and the Like and Comment tools (on the main account) and then let them run, make sure to avoid aggressive settings and make sure to stay below 200 Like and 30 comments per day on the main account.
