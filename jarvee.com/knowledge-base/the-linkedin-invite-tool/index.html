The Linkedin Invite Tool

LinkedIn is not only an excellent platform to grow your professional network. It is also a good venue to promote your business through pages and groups.

With the Jarvee LinkedIn Invite tool, you can automatically invite your connections to join your groups as well as to follow your pages.

How to access LinkedIn Invite Tool?

To access the LinkedIn Invite tool you need to go to TOOLS page > click on your LinkedIn Profile > then click on the Invite tool from the tools menu on the left.

There are two tabs in the LinkedIn Invite Tool: Add Friends to Groups and Invite Connections to Follow Pages.

Add Friends to Groups

In this sub-tab, there are two sections: Run Timers and Invite Friends to Groups Settings.

Run Timers section:

  1. Wait between X and Y seconds before each new operation – this is the number of seconds you want the tool to wait before executing a new invite operation.
  2. Invite between X and Y people per operation – allows you to set the number of people that are going to be invited during each operation.
  3. Execute between HH: MM and HH: MM – indicates the execution time of the invite tool. Aside from setting the timer, you can also add more intervals or set Jarvee to randomize the timers daily.
  4. Days of the week when to operate – allows you to select the days when the tool will perform the invite actions.

💡  If you select the option “Rotate Days”, Jarvee will use the same number of days you have initially selected and rotate them.

Example: You selected Monday, Wednesday, and Friday.  On the following week, Jarvee will choose a new set of 3 days randomly. You may also exclude weekends in the variation if you prefer not to invite your friends to your groups on weekends.

Invite Friends to Groups Settings section:

  1. Invite X – Y per day – Set the number of friends you want to invite to groups per day.

A random number will be picked between X-Y and will be used as the daily limit for that day.  You can also set the Invite tool to gradually increase the daily limit each day until it has reached your preferred target limit.

Example:

If you set the Invite tool daily limit like the limit above, Jarvee will pick a random number between 40 and 50 as a daily limit each day. The daily limits will increase gradually by the random value chosen between 4 and 8 each day until the daily limit reaches 70 per day.

  1. Invite X-Y per hour – allows you to set the number of friends you want to invite per hour. A random number between X and Y will be chosen as the hourly limit. There’s an option to increase the hourly limit gradually as well.
  2. Auto Select New Groups – if you have this option checked, you don’t need to select the new groups manually. After the sync operation, the groups that are added to the table will be selected automatically.
  3. Auto Select New Friends – If this option is checked, the newly added connections/friends will automatically be selected to be invited to the selected groups in the Groups table.

Groups Table

The table shows LinkedIn groups you’re already a part of and the number of friends/connections that have been invited to the group.

💡 If you have already joined a group, but it doesn’t show up on the group table, you need to Sync your LinkedIn account from the Social Profiles tab (Actions > Sync).

Under the Groups table you will find six options:

1. Advanced Selection – if you click on this option, the Advanced Selection window will pop up. You can use the filter feature or the tags filter feature to find your preferred groups quickly. This is useful when you’re a member of many groups.

2. Select All – clicking on this option will select all groups shows in Groups table

3. Select None – no group is selected if you click on this option.

4. Reset Group Invites – this option will reset the number of invited friends to 0.

5. Export Selected – this allows you to export the selected groups into a CSV file.

6. Import Groups – this is useful when you have already a list of LinkedIn groups URL you want to invite your friends to. The import file must follow the same format as the export file. You can export one of your groups to view the correct CSV format.

Friends Table

All your friends/connections are shown in this table. Only the selected friends are going to be invited to the chosen groups in Groups Table.

There are three options under the table:

1. Force Friends Update – allows you to force Jarvee to sync your friends. After the sync operation has finished, you will see your new friends at the table. Another way to update your friends list is by Syncing your profile from the Social Profiles tab (Actions > Sync).

2. Select All – helps you select all your friends in the table.

3. Select None – none of your friends will be selected if you click on this option

The total number of friends invited to the chosen groups will be shown below the table as well.

Invite Connections to Follow Pages

If you want to invite your connections to follow your company/business pages automatically, this is your go-to Invite sub-tab.

There are two sections in this tab: Run Timers and Settings section.

Run Timers section:

  1. Wait between X and Y seconds before each new operation – Jarvee will choose a random number between X and Y as wait time before each new operation.
  2. Invite between X and Y people per operation – indicates the number of connections/friends that are going to be invited to follow pages during each operation. A number between X and Y will be chosen randomly on each operation.
  3. Execute between HH: MM and HH: MM – allows you to adjust the execution time of the invite tool. You can also add more intervals and/or opt to randomize the intervals daily.
  4. Days of the week when to operate – the tool will perform the invite actions only on the selected days.

💡 If you select the option “Rotate Days”, Jarvee will use the same number of days you have initially selected and rotate them.

Example: You selected Monday, Wednesday, and Friday.  On the following week, Jarvee will choose a new set of 3 days randomly. You may also exclude weekends in the variation if you prefer to send the invites only on weekdays.

Settings section:

  1. Invite X – Y per day – indicates the number of connections that are going to be invited to the chosen pages.

A number between X-Y will be picked randomly and be used as a daily limit for that day. You can also set the Invite tool to gradually increase the daily limit each day until it has reached your preferred target daily limit.

Example:

If you have the above settings as your daily limit, Jarvee will pick a random number between 50 and 60 as a daily limit each day. The daily limits will increase gradually by the random value chosen between 5 and 10 each day until the daily limit reaches 100 per day.

  1. Invite X-Y per hour – allows you to set the number of connections you want to invite per hour. A random number between X and Y will be chosen as the hourly limit. There’s also an option to increase the hourly limit gradually just like the daily limit.
  2. Auto Select New Pages – if you have this option checked, you don’t need to select manually the pages you want your connections to follow.
  3. Auto Select New Friends – the new connections/friends will be invited automatically to the selected pages in the Pages table if you check this option.

Pages Table

All pages you have on your LinkedIn account will be displayed in this table. When you run the invite tool, Jarvee invites the chosen connections/friends to the selected pages.

💡 If your new page is not shown in the table, you should sync your LinkedIn profile from the Social profiles tab.

There are three options under the pages table:

1. Select All – allows you to select all pages

2. Select None – none of your pages will be selected

3. More Actions – a new window will pop up if this option is clicked. There are four options you can use:

  • Reset page invites – this will reset the number of invited friends to 0.

  • Select chosen pages/events in all other same tag accounts – if you have multiple LinkedIn accounts with the same tag name and you want to invite those accounts’ connections to the same pages, you can click on this option. The same pages on those accounts will be selected as well.
  • Select all pages/events in all same tag accounts – allows you to select all pages/events on other LinkedIn accounts that have the same tag name.
  • Sync chosen pages/events in all other same tag accounts– this will update the chosen pages/events on all other LinkedIn accounts that have the same tag name. They will all have the same chosen pages as the account.

Friends Table

There are four options under the Friends table:

1. Force Friends Update – clicking on this option will force Jarvee to sync your friends. If there’re new friends, they will be added automatically to the table.

2. Select All – all friends in the table will be selected if you click on this option.

3. Select None – none of your friends will be selected.

4. Select All Friends in All Same Tag Accounts – you can click on this option when you have other LinkedIn accounts that have the same tag name and you want to select all friends on those accounts as well.

The total number of friends that have been invited to pages will be displayed below the Friends table.
