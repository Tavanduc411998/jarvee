The Invite Friends Tool

One of the best tactics to boost your social media presence is to invite friends to groups and pages you either own or manage but this could be very time consuming and results to being counterproductive.  Hence, to maximize your time, you can use JARVEE’s Invite Tool which is available to Facebook and LinkedIn only.

The Invite Tool automatically adds friends to Groups and invites friends to your Pages.  In addition to that, it can also invite non-friends to your Pages by inviting those who liked the posts of the said pages.

✅ You are not limited to only inviting to Groups and Pages you own or manage.  You can add friends to Groups you’ve joined and to other Pages that you’ve added manually.

How to access the Invite Tool

To begin using the tool, go to the Tools page and click on the Facebook or LinkedIn account where you want to run the tool on or click on the MORE TOOLS link then navigate to the Invite tab.

At first glance on the Facebook Invite tab, you will see that it is subdivided into four tabs

  1. Add Friends to Groups
  2. Add Emails to Groups
  3. Invite Friends to Pages/Events
  4. Invite From Page Post Likes

💡 Click 👉 here if you wish to know more about Linkedin  Invite Tool.

Add Friends to Groups (for Facebook and LinkedIn)

If you want to grow your groups, you go to this sub-tab.  This is where you can select the friends you want to add to your groups.  The Add Friends to Groups tab is comprised of 4 main parts; Run Timers, Invite Friends to Groups Settings, Groups Table and Friends Table.

Run Timers

This is the part where you control the behavior of the tool.  Here you can:

  1. Configure the intervals between each new operation.

✅  ‘Operation’ refers to each set of ‘add friends to groups’ action.

  1. Select the number of people that would be added per each operation
  2. Designate the in-between hours when the tool is going to add friends to your groups

💡 On rotating days, JARVEE would pick the equivalent number of days you have initially selected; the selection is random and you have the option to exclude weekends in the variation in case you don’t want any ‘add friends to groups’ activity on Saturdays and Sundays.

Invite Friends to Groups Settings

This portion of the settings is where you specify the extent and limitation of the tool.

  1. You can choose the total number of friends you want to add to your groups per day and you can opt to gradually increase that number until it hits the maximum number you favor.

✅ You can set the ‘Invite maximum X-Y per day’ to ‘ 0 ‘ for no limit.

  1. You can choose the total number of friends you want to add to your groups per hour and you can opt to gradually increase that number until it hits the maximum number you favor.
  2. If you are also running the Joiner tool on the account, you may set the tool to ‘Auto Select New Groups’ by ticking the equivalent checkbox.
  3. Aside from automatically selecting the new groups, you can also check the option ‘Auto Select New Friends’ and just like what it does with new groups, it would automatically select your new friends.

✅ Both ‘Auto select…’ options mentioned on 6-7, would enable you to enjoy the full benefits of running the tool in auto, so it’s highly recommended that you check them.

Groups Table

 

You will see all your groups on the Groups table.  These are the groups that you can select and add friends to.

The Groups Table has 3 headers:

  • Selected – below it, you will see a checkbox, simply tick to select the respective group on the list
  • Name – this is the name of the group
  • #Invited – is the number of friends that have been added to the particular group

Below the Groups Table, you’ll see a couple of action links:

Most are self-explanatory like Select All, Select None, Export Selected and Import Groups.

You also have:

Reset Group Invites – when clicked, the number listed under #Invited would change to ‘ 0 ‘

Advanced Selection – when you click on this action link, a new window will appear and from the said window, you can filter the groups where you want to add friends to.  This would be effectual when you have hundreds of groups and you only want to select those who have your target minimum or maximum number of users or with designated tag

💡 When you have finished selecting the groups by checking the appropriate checkbox, you can quickly execute actions on them by clicking your favored action from the links below or you can click the ‘Execute Actions On Selected Items’ drop down menu and clicking your desired action there.

Friends Table

On this table, you can pick the friends you want to add to your chosen groups.

  • Selected – all you need to do is tick the checkbox at the left side of your friend’s name
  • Name – the name of the Facebook profile

At the bottom of the Friends Table, you will see the action links Select All, Select None and Force Friends Update – you can quickly update your friends list by clicking on this action link.

 ✅ To start adding your friends to groups, just switch the Status of the tool to ON.

💡 If you have a lot of Facebook accounts where you want to run the same tool on plus implement the exact same settings, you can click on the COPY SETTINGS button at the upper portion of the page.

Add Emails to Groups

If you have a list of user emails, another way to grow your groups is to Invite users to your groups using their emails, you can go to this sub-tab.  This is where you can add Facebook user emails that you want to invite to your groups.  The Add Friends to Groups tab settings consist of 4 main parts; Run Timers, Invite Friends to Groups Settings, Groups Table and Add emails to Invite table.

Run Timers

This is the part where you control the behavior of the tool.  Here you can:

  1. Configure the intervals between each new operation – You can change wait between x and y seconds before each operation.
  2. Set the number of emails you want to invite per operation.
  3. Indicate between your preferred hours when the tool executes as well as having the option of adding Intervals and you can also randomize intervals daily.
  4. You can specify the days of the week when to operate and you can also rotate between days. 

Invite Emails to Group Settings

  1. You can specify how many emails to invite to Groups per day and you can gradually increase that number until it hits that specified target number.
  2. Here you can specify how many emails to invite to Groups per hour and you can gradually increase that number until it hits that specified target number.
  3. You may set the tool to ‘Auto Select New Groups’ by ticking the assigned checkbox.

Groups Table

 

Looking at the Pages Table, you have

  • Selected – you have a tick box below it, just check to select the Page/s
  • Name – this is the name of the Page
  • #Invited – is the number of friends that you have invited to that specific page

Below the Pages Table, you’ll see some action links like Select All, Select None and

Reset Page Invites –when clicked, the number listed under #Invited would change to ‘ 0 ‘

Add emails to Invite Table

  • Input the emails in the ‘Add Emails to invite’ field then click ‘Add to Email List’ button.
  • After Clicking the ‘Add to Email List’ button, you should now be able to see the emails added on the Index table below.

At the bottom of the Friends Table, you will see the action links Select All, Select None and Force Friends Update – you can easily update your friends list by clicking on this action link.

💡 If you have a lot of Facebook accounts where you want to run the same tool on plus implement the exact same settings , you can click on the COPY SETTINGS button at the upper portion of the page.

Invite Friends to Pages/Events
Aside from adding friends to groups, you can also auto invite friends to Pages/Events you own, manage or other pages that you are not associated with. And to accomplish that goal, simply go to the Invite Friends to Pages tab.

Just like the ‘Add Friends to Groups’ tab, it also has: Run Timers, Invite Friends to Pages Settings, Pages Table and Friends Table. You can configure it the same way you did when you added friends to groups.

Run Timers

This is the part where you control the behavior of the tool.  Here you can:

  1. Configure the intervals between each new operation.

✅ ‘Operation’ refers to each set of ‘invite friends to pages’ action.

  1. Indicate the in-between hours when the tool is going to invite friends to the selected pages
  2. Choose the days when the tool is going to execute the action. You can rotate the days and exclude weekends if you want.

💡 On rotating days, JARVEE would pick the equivalent number of days you have initially selected; the selection is random and you have the option to exclude weekends in the variation in case you don’t want any ‘invite friends to pages’ activity on Saturdays and Sundays.

Invite Friends to Pages/Pages Settings

  1. You can choose the total number of friends you want to invite to the selected pages per day and you can opt to gradually increase that number until it hits your maximum target number.
  2. You can choose the total number of friends you want to invite to the selected pages per hour and you can opt to gradually increase that number until it hits your maximum target number.

✅ You can set the ‘Invite maximum X-Y per day’ to ‘0‘ for no limit.

  1. You may set the tool to ‘Auto Select New Pages’ by ticking the assigned checkbox.
  2. In addition, you can also check the option ‘Auto Select New Friends’ and the same with new pages, it would automatically select your new friends.
  3. If you want to invite all your friends in one go you can check the option ‘Invite all friends for pages in one go’.

✅ Both ‘Auto select…’ options mentioned on 6-7, would enable you to take advantage of running the tool in auto, so it’s highly recommended that you check them.

Pages Table

Looking at the Pages Table, you have

  • Selected – you have a tick box below it, just check to select the Page/s
  • Name – this is the name of the Page
  • #Invited – is the number of friends that you have invited to that specific page

Below the Pages Table, you’ll see some action links like Select All, Select None and More Actions:

Reset Page Invites –when clicked, the number listed under #Invited would change to ‘0‘

Friends Table

On this table, you can pick the friends you want to invite to your pages.

  • Selected – all you need to do is tick the checkbox at the left side of your friend’s name
  • Name – the name of the Facebook profile

At the bottom of the Friends Table, you will see the action links Select All, Select None and Force Friends Update – you can easily update your friends list by clicking on this action link.

Select all Friends in all Same Tag Account – this will only select all friends in all the same tag account.

  1. To start inviting friends to your pages, just switch the Status of the tool to ON.

💡 If you have a lot of Facebook accounts where you want to run the same tool on plus implement the exact same settings , you can click on the COPY SETTINGS button at the upper portion of the page.

Invite From Page Post Likes

This is the Invite tool where you can invite non-friends who liked the posts of your pages to like your page as well.  Similar to the two other invite tabs, you have Run Timers, Invite From Page Posts Like Settings and Pages Table

Run Timer

  1. Configure the intervals between each new operation.

✅ ‘Operation’ refers to each set of ‘invite to page’ action.

  1. Select the number of people that you want to invite per operation
  2. Specify the in between hours when the tool is going to invite people to the selected pages
  3. Choose the days when the tool is going to execute the action. You can rotate the days and exclude weekends if you want.

💡 On rotating days, JARVEE would pick the equivalent number of days you have initially selected; the selection is random and you have the option to exclude weekends in the variation in case you don’t want any ‘invite people to pages’ activity on Saturdays and Sundays.

Invite From Page Posts Like Settings

  1. You can choose the total number of people you want to invite to the selected pages per day and you can opt to gradually increase that number until it hits the maximum number of invites you are eyeing.
  2. You can choose the total number of people you want to invite to the selected pages per hour and you can opt to gradually increase that number until it hits the maximum number of invites you are eyeing.

✅ You can set the ‘Invite maximum X-Y per day’ to ‘0‘ for no limit.

  1. You can choose to send the invitation to like to the last X posts from the Page
  2. Set the maximum seconds the tool will scroll on the likers of the post
  3. You can opt to stop the invite operation after X minutes
  4. You can also auto-select New Pages by checking this box.
Pages Table

Looking at the Pages Table, you have

  • Selected – you have a tick box below it, just check to select the Page/s
  • Name – this is the name of the Page
  • #Invited – is the number of friends that you have invited to that specific page

Below the Pages Table, you’ll see some action links like Select All, Select None and

Reset Page Invites –when clicked, the number listed under #Invited would change to ‘ 0 ‘

Specify Facebook Page Permalinks to Process – Here you can add Facebook Page URLs.

✅ To start inviting people who liked the posts to also like the page, just switch the Status of the tool to ON.

💡 If you have a lot of Facebook accounts where you want to run the same tool on plus implement the exact same settings , you can click on the COPY SETTINGS button at the upper portion of the page.
